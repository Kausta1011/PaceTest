# PaceTest

Master's thesis project studying agent-tool co-adaptation cycling.

## Setup

1. Install Ollama from ollama.com
2. Pull the model: `ollama pull qwen3:8b`
3. Install Python dependencies: `pip install -r requirements.txt`
